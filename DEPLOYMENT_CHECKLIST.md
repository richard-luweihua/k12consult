## 部署命令参考

```bash
# 本地测试
npm install
npm run dev

# 本地构建测试
npm run build
npm run start

# Git 推送到 GitHub
git push origin main

# Vercel 部署（通过 GitHub 自动触发）
# 或在 Vercel 仪表板手动部署
```

## 紧急回滚

如果生产环境出现问题：

1. 在 Vercel 仪表板找到上一个稳定的部署
2. 点击 "Redeploy"
3. 验证功能恢复正常
4. 记录问题并进行根因分析
